# The Hitchhiker's Guide to Python

A best practice handbook for Python installation, configuration, and usage, built with [mdBook](https://rust-lang.github.io/mdBook/).

**Read it here: https://jestrada.github.io/python-hitch-guide/**

## Building locally

```bash
mdbook serve --open
```

## Deployment

Pushes to `main` are deployed to GitHub Pages via `.github/workflows`.
